## JavaDevelopment

This repository contains a collection of Java practice projects, exercises, and mini-applications developed while learning Core Java, OOPs, JDBC, Servlets, Spring Core, and more.

Each folder represents a separate concept, module, or small project created during hands-on learning.

---

## Repository Structure

### 1. OOPs & Core Java
- **Abstract** – Programs demonstrating abstract classes and methods  
- **Collection** – Java Collections Framework examples (List, Set, Map, etc.)  
- **Multithreading** – Thread creation, synchronization, lifecycle  
- **Oops** – Object-Oriented Programming concepts  
- **SessionDemo** – Session management basics  

---

### 2. Web & Server-Side
- **FirstWebApp** – Basic Java web application using Servlets/JSP  
- **ServletLifeCycle** – Servlet lifecycle demonstrations  
- **HttpServerDemo** – Basic HTTP server implementation  
- **HTML** – Static HTML files used for testing  

---

### 3. JDBC & Database
- **JDBC01** – Basic JDBC examples  
- **JDBCDemo** – CRUD operations, Statement & PreparedStatement  
- **JDBCP01** – Additional JDBC practice programs  

---

### 4. Dependency Injection / Spring
- **InjectionDemo** – Constructor and Setter Injection  
- **InjectionDemo1** – Additional DI examples  
- **Spring Core** – Spring IoC container, Beans, Autowiring  

---

### 5. Mini Projects
- **BookMyTrain** – Train booking console project  
- **GrageBilling / GrageBillingSystem** – Billing system project  

---

### 6. Tools & Servers
- **apache-tomcat-11.0.14-windows-x64** – Tomcat server used for running servlet projects  

---

## How to Run

### Core Java Programs
```
javac FileName.java
java FileName
```

### Servlet-Based Projects
1. Install Apache Tomcat  
2. Deploy the project inside the `webapps` folder  
3. Start Tomcat  
4. Access in browser:  
```
http://localhost:8080/ProjectName
```

### Spring Core Projects
- Open in IntelliJ/Eclipse  
- Add required Spring JARs  
- Run the main class
